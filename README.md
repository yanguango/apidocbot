# Apidocbot

A simple bot to generate api docs

## Installation

Add this line to your application's Gemfile:

    gem 'apidocbot'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install apidocbot

## Usage

	rails generate apidocbot
	write your api doc yaml files
	rake apidocbot
